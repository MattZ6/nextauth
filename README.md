## 🎉 Começando

Clone o repositório:

```bash
git clone https://github.com/MattZ6/nextauth
```

Adentre a pasta do projeto:

```bash
cd nextauth
```

Instale as dependências:

```bash
yarn
```

## 🔥 Executando

Para executar a aplicação em ambiente de desenvolvimento (localmente):

```bash
yarn dev
```
